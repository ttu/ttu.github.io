Youtube to embedded flash (page)

  • With this you can watch youtube videos in places where youtube's site is blocked. (Not guaranteed to work in all places)
  • Insert url of youtube video to textbox and choose size.
  • Page creates new embedded flash object.

  • http://www.youtube.com/watch?v=am76xV8Aubk is normal youtbe web page
  • http://www.youtube.com/v/am76xV8Aubk is url to video. (if you use this url, it opens it automatically to embedded flash, but in fullscreen)
  • Javascript replaces youtube.com/watch?v= with youtube.com/v/ from url that is used for video.
